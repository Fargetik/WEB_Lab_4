## ⚙️ Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Fargetik/WEB_Lab_4
   cd WEB_Lab_4
   
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
3. Открытие главной php страницы:
```http://localhost:8080/index.php```

4. Открытие формы:
```http://localhost:8080/form.html```

5. Просмотр содержимого файла data.txt:
```http://localhost:8080/view.php```

---

## 📂 Содержимое проекта

```docker-compose.yml``` — описание сервиса Nginx

```www/index.php``` — PHP-страница. Показ данных текущей сессии и ошибок

```www/process.php``` - Процесс сохранения данных и поиск ошибок

```www/view.php``` - Показ сохранённых данных через файл

```www/form.html``` - HTML-Форма

```www/APIClient.php``` - Файл для запроса из публичных API

```www/UserInfo.php``` - Показ Информации о пользователе
